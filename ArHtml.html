En esta unidad hemos visto comandos muy utiles para el manejo de Git.
Estos son algunos de ellos:
-cd : Sirve para moverse entre directorios.

- git init: Para iniciar un repositorio remoto.

- git status: Para ver el estado del repositorio.

- git add: Para incluir los archivos o directorios a la siguiente version.

- git commit: Sirve confirmar las modificaciones y crear una nueva version con los archivos en seguimiento.

- git push: Este se usa para sincronizar nuestro repositorio local con el repositorio remoto.

- git log: Funciona para cuando queremos ver el historia de nuestro repositorio local en caso de querer recueprar una version anterior.
